### Prerequisites
- Git
- Node js
- Ionic

- Firebase (```npm install -g firebase-tools```)

### Prepare

```shell
$ git clone https://github.com/samuele-cozzi/Omarks3.git
$ code .\Omarks3\
$ npm install
$ cd functions
$ npm install
$ cd ..
```

### Execute

```shell
$ ionic build
$ ionic serve
```

### Delivery

```shell
$ ionic build
$ firebase deploy
```
